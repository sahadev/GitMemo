## 2. 视频简介（推荐中文版）

我做 GitMemo，是因为我一直在丢失那些本来很有价值的 AI 对话。

平时我会用 Claude Code 和 Cursor 做调试、写脚本、做设计、拆计划。真正有用的内容其实已经在对话里产出了，但一旦会话结束，它们就散落在历史记录、剪贴板和零散笔记里。过一段时间，我又得重新问一遍同样的问题。

GitMemo 的目标很简单：把这些 AI 对话、笔记和日常工作内容，沉淀成一个由 Git 管理、归你自己所有的知识库。

它目前支持：
- Claude Code
- Cursor
- 本地优先的 Git 知识库保存
- CLI、Desktop、MCP 三种入口
- 全文搜索
- 笔记、日记、文档管理
- 可选的剪贴板捕获

适合谁？
- 经常使用 Claude Code / Cursor 的开发者
- 想把 AI 产出沉淀下来的人
- 不想把内容锁进另一个云端 SaaS 的人
- 希望自己的知识库是本地可控、可搜索、可版本化的人

如果你也有“AI 聊天很有价值，但过几天就找不到了”的问题，GitMemo 可能会对你有帮助。

官网：https://git-memo.vercel.app/
GitHub：https://github.com/sahadev/GitMemo
Product Hunt：可在发布后补上链接

## 4. 英文视频简介

I built GitMemo because I kept losing useful AI conversations I wanted to reuse later.

GitMemo helps Claude Code and Cursor users save AI conversations, notes, and everyday work into a Git repo they own. It captures useful outputs as searchable Markdown in Git, so you can find them later from CLI, Desktop, or MCP.

Key points:
- Local-first
- Remote Git sync optional
- Built-in notes, daily notes, and docs
- Full-text search
- Optional clipboard capture in Desktop (macOS)
